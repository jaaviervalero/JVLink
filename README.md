# JVLink 🔗

A high-availability, Cloud-Native URL shortener built for speed and minimal resource consumption.

## 🚀 Tech Stack

- **API Engine:** Python (FastAPI)
- **Database:** PostgreSQL
- **Infrastructure:** Proxmox LXC

_Currently under construction. Just a MVP._
